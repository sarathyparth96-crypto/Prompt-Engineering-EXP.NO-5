EXP 5: COMPARATIVE ANALYSIS OF DIFFERENT TYPES OF PROMPTING PATTERNS AND EXPLAIN WITH VARIOUS TEST SCENARIOS

Aim: To test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. Analyze the quality, accuracy, and depth of the generated responses.

Basic prompting:
A basic prompt is a clear and structured instruction given to the AI to guide it in producing accurate and relevant results.
Different prompting patterns help improve:
Accuracy


Clarity:


1.Depth of response


2.User understanding


3.This experiment compares multiple prompting patterns using test scenarios.


Types of prompt:
1. Straightforward Prompt:

A direct and simple prompt that asks the AI to perform a specific task.

Example Prompt:

"Explain what Artificial Intelligence is."

Output (Example):

Artificial Intelligence (AI) refers to the simulation of human intelligence in machines that are programmed to think, learn, and solve problems.

Characteristics:

Simple and easy to use

Quick responses

Less detailed output

2. Comparative Analysis Prompt:

This prompt asks the AI to compare two or more concepts.

Example Prompt:

"Compare Artificial Intelligence and Machine Learning."

Output (Example):
```
| Aspect     | Artificial Intelligence                      | Machine Learning                                     |
| ---------- | -------------------------------------------- | ---------------------------------------------------- |
| Definition | Broad field of creating intelligent machines | Subset of AI that allows machines to learn from data |
| Scope      | Large                                        | Narrow                                               |
| Examples   | Robotics, Expert systems                     | Recommendation systems                               |
```
Characteristics:

Helps in understanding differences

Produces structured responses

Useful for research and learning

3. Experiential Perspective Prompt:

This prompt asks the AI to respond from a specific perspective or role.

Example Prompt:

"Explain Artificial Intelligence from the perspective of a software engineer."

Output (Example):

As a software engineer, AI can be seen as a tool that enables systems to perform intelligent tasks such as decision-making, pattern recognition, and automation.

Characteristics:

Context-based responses

More engaging and realistic

Useful for simulations

4. Tabular Format Prompt:

The prompt specifically requests information in table format.

Example Prompt:

"List advantages and disadvantages of Artificial Intelligence in a table."

Output (Example):
```
| Advantages           | Disadvantages       |
| -------------------- | ------------------- |
| Automation           | Job displacement    |
| Fast decision making | High cost           |
| Error reduction      | Data privacy issues |
```
Characteristics:

Organized output

Easy comparison

Good for reports

4. Test Scenarios:

To analyze the effectiveness of prompting patterns, the following scenarios were tested.

Scenario 1: Concept Explanation
Prompt:

"Explain Blockchain Technology."
```
| Prompt Type            | Observation                                    |
| ---------------------- | ---------------------------------------------- |
| Straightforward Prompt | Gives simple explanation                       |

```

"Compare Cloud Computing and Edge Computing."
```
| Prompt Type     | Result                      |
| --------------- | --------------------------- |
| Straightforward | Basic explanation           |

```
Scenario 3: Problem Solving
Prompt:

"Explain how AI can be used in healthcare."
```
| Prompt Type     | Observation          |
| --------------- | -------------------- |

```
